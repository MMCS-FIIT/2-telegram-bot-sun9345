# Бот для получения рекомендаций в музыке

- /add или прислать трек в формате .mp3 для добавления трека в библиотеку
- /list для просмотра библиотеки
- /delete [номер] для удаления трека из библиотеки
- /recommend выдает похожих исполнителей на автора рандомного трека из библиотеки

### Библиотека хранится локально в .csv формате 
### Может считывать mp3 теги с присылаемых файлов
### Рекомендации строятся через похожих исполнителей на сервисе скробблинга музыки Last.fm
